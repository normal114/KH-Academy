변수의 사용

변수 선언 규칙
1.대소문자 구분
2.영문, $, _ 로 시작
3.영문, $, _, 숫자를 포함
4.키워드나 예약어를 사용 불가
변수의 범위
1.전역변수 : window 객체에 포함되는 변수로, 다른 함수들에서 공용으로 사용 가능 (값이 유지됨)
2.지역변수 : 함수나 객체 내부에 선언되고, 실행이 종료되면 사라짐
변수의 선언과 저장
1.var 키워드를 사용하여 변수를 선언
2.변수의 타입은 저장되는 타입에 따라 결정
타입의 종류
문자(string)
숫자(number)
논리(boolean)
널(Null)
객체(object)
