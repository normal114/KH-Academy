Program : 실행파일
Process : 실행되고 있는 상태
Thread  : 작업 단위

Process (Runtime) : 응용프로그램(.exe) 단위로 실행되는 모듈
Runnable (Thread) : 하나의 process에 여러 개의 메소드 단위로 실행되는 모듈

Process - T, T, T, ...
(process : os에서 제어할 수 있는 가장 작은 단위)

* 자바에서 메소드 단위로 프로그램을 실행한느 방법 2가지
1. Runnable 을 impliments (간단하게 실행할 때)
2. Thread 를 extends (기능이 여러 개가 필요할 때) -> Thread 간 양보, 죽이기 등등..

* run() 메소드는 Thread.start()가 자동 호출
